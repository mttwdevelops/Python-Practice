To run the files, we need some things.

# It is important that we make our code as useable on any machine
#   as possible, which is why we will use a virtual environment

# venv set up by cd'ing to cd and using python3 -m venv .venv

# venv activated using: source .venv/bin/activate
# venv deactivated using: deactivate

# upgraded pip in venv env using pip install --upgrade pip

To install packages:
pip install -r requirements.txt

To check packages:
pip list
